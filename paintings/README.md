
# Wall Paintings

Place your painting images here. The game will automatically load them onto the restaurant walls.

## Required filenames (JPG format):

- `painting_01.jpg`
- `painting_02.jpg`
- `painting_03.jpg`
- `painting_04.jpg`
- `painting_05.jpg`

## Recommended content:
- Filipino landscapes (rice terraces, Mayon Volcano, Chocolate Hills)
- Jeepney artwork
- Kalabaw / carabao paintings
- Bahay kubo scenes
- Filipino food illustrations

## Size: 512×512 or 1024×512 pixels recommended.

If no images are placed here, the frames will show a warm fallback color automatically.
